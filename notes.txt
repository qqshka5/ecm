[definition] позиционированный элемент (positioned element): любой элемент со свойством position отличным от static
позиционированный элемент становится системой координат для его потомков с position: absolute
на позиционированные элементы действует св-во z-index

[definition] viewport: доступная для просмотра документа область браузера

структура описания св-в позиционирования:
- занимает ли элемент место в нормальном потоке
- как определяются размеры элемента
- как определяется сиситема координат для позиционирования элемента

[definition] нормальный поток (normal flow):
расположение элементов в основном слое документа следующим образом:
- блочные элементы занимают всю доступную ширину, высота опр содержимым, если не указана явно
- строчные элементы располагаются друг за другом по горизонтали, при достижении правой границы содержащего блока
образуется новая строка, размер строчных элементов определяется их внутренними характеристиками (например, размер шрифиа,
кол-во текста), свойсва width and height не действуют для текста
